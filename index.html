Darkest Dungeon

Darkest dungeons user interface is somewhat remarklable, to have so much information in a easy to see and use layout is interesting. Dedicating 1/3 of the screen to it's UI may seem like a bad idea but having so many items such as names, abilities, health, stats, gear, items and a map all on this UI it does so beautifully.
